## 📖 Overview

FinRelief AI is a full-stack web application that helps individuals better understand and manage their financial obligations. The platform enables users to securely manage loan information, assess financial health through key metrics, and receive AI-generated recommendations for debt negotiation and settlement.

The project combines a modern React frontend with a FastAPI backend and integrates Google's Gemini AI to provide intelligent financial assistance.

# 🚀 Getting Started

## Prerequisites

Install the following before running the project:

- Python 3.10+
- Node.js 18+
- npm
- Git

---

## Clone Repository

```bash
git clone https://github.com/datta000000000/FinReliefAI.git

cd FinReliefAI
```

---

## Backend Setup

```bash
cd backend

python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run backend

```bash
uvicorn app.main:app --reload
```

Backend URL

```
http://localhost:8000
```

---

## Frontend Setup

Open another terminal.

```bash
cd frontend

npm install

npm run dev
```

Frontend URL

```
http://localhost:5173
```
